# Valve Server Query
## Install
### Maven
```xml
<dependency>
    <groupId>io.github.anominy</groupId>
    <artifactId>valve-server-query</artifactId>
    <version>0.1-r0</version>
</dependency>
```
### Gradle
```groovy
implementation 'io.github.anominy:valve-server-query:0.1-r0'
```
